# Parking

## Description
Application web permettant d’attribuer automatiquement des places de parking aux membres du personnel, de gérer les réservations et la file d’attente.


## 🛠️ Technologies
- **Backend** : PHP, MySQL
- **Frontend** : HTML5, CSS3 (responsive)
- **Serveur** : Apache
